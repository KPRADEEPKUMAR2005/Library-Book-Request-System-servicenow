# Library Book Request System

## Overview

The Library Book Request System is a ServiceNow application that allows users to request library books online. The system helps librarians manage book requests, approvals, and book records efficiently.

## Features

* Book Request Submission
* Request Approval Workflow
* Book Inventory Management
* Status Tracking
* Notifications
* Reports and Dashboards

## Technologies Used

* ServiceNow
* Service Catalog
* Flow Designer
* Business Rules
* Custom Tables
* Reports & Dashboards

## Workflow

1. User submits a book request.
2. Request is sent for librarian approval.
3. Librarian approves or rejects the request.
4. User receives status updates.

## Benefits

* Reduces manual work
* Improves request tracking
* Centralized library management
* Faster approval process
